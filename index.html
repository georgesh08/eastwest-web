О нас

О нас

Всем привет! Рады приветствовать Вас в нашем международном языковом коммьюнити East&West Language Club. Клуб проводит офлайн разговорные занятия с носителями языка в Санкт-Петербурге, а также обеспечивает онлайн занятия с доступом из любой точки мира.
Наши встречи проводятся в различных форматах: дискуссии, просмотр кино/видео, настольные игры, вечеринки, знакомства и многое другое. Присоединяйся к нам, достигай новых языковых высот и стань частью большой международной семьи!

public static int PowerCounter(int time, Knowledge knowledge)
{
 return time * knowledge.value;
}

public static void Main(string[] args)
{
 Knowledge k = new Knowledge();
 k.value = 10000;
 Console.Writeline(PowerCounter(12, k));
}
